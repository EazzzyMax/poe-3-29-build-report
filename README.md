# Path of Exile 3.29 — Build Market Report

Персональный русскоязычный отчёт по рынку league starter-билдов для Path of Exile 3.29: Curse of the Allflame.

- Практический рейтинг билдов с плюсами, минусами и рисками.
- Источники опубликованы или обновлены после анонса 16 июля 2026 года.
- Для каждого билда сохранены прямые ссылки на видео, гайды и обсуждения.
- Под поиском есть визуальный фильтр по аскенданси: 19 локальных портретов сгруппированы по семи базовым классам, а варианты без билдов в текущем рейтинге показаны отключёнными.
- В карточках показаны локальные портреты ascendancy и цельные квадратные иконки main skill из PoE Wiki; страница не зависит от внешнего hotlink изображений.
- В шапке доступны четыре независимых среза: исходный `GPT version`, повторные аудиты `GPT version 2` и `GPT version 3`, а также авторский порядок `Fubgun version`.
- `GPT version 2` сохраняет старый рейтинг неизменным, но отдельно пересчитывает 26 билдов после финальных правок и гайдов 22 июля; над таблицей есть разворачиваемый changelog.
- `GPT version 3` — финальный предзапусковой срез 24 июля на 32 билда: добавлены свежие тирлисты Spicysushi и Ruetoo, новые гайды Zizaran / Goratha / BalorMage / PhazePlays, проверены PoB из описаний и обновлены Reddit-сигналы. Над рейтингом сохранён подробный changelog относительно v2.
